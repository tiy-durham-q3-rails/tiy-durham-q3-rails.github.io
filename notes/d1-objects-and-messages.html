Objects and Messages¶

Everything in Ruby is an object. What does that mean?

Objects have:

  • state
  • behavior

Out[1]:

You cause objects to act by sending them messages which they respond to.

Out[2]:

In [3]:

Out[3]:

A spoonful of sugar

object.send :message is pretty wordy, and it makes it really hard when we want to send a message to the result of a previous message.

Syntactic sugar is a term that means syntax in a language that makes things easier to read or write. Ruby has some sugar for sending messages that makes it much nicer. You can write object.message. In fact, you've been doing this with object.send.
