Starting Ruby

From the command-line, run irb.

IRb stands for "Interactive Ruby."

Ruby values

Ruby has a few basic value types we should know in order to use it.

  • integers (fixnums)
  • decimals (floats)
  • strings
  • symbols
  • booleans
  • arrays
  • hashes

What is a hash?

A hash is an unsorted collection of keys and values.

Instead of numbers pointing to each value, there's a "key" or a name.

In [46]: 
person = {:first => "Clinton", :last => "Dreisbach", :hometown => {:city => "Durham", :state => "NC"}}
person[:hometown][:state]
Out[46]:
"NC"
In [31]: 
person[:last]
Out[31]:
"Dreisbach"
In [32]: 
person[:middle]

You don't have to use symbols for keys. Other things work, too.

In [44]: 
song = {"key" => "G", "chord_progression" => [1, [1, 2, 3], 5]}
song["chord_progression"][1][2]
Out[44]:
3

Symbols are used so often for keys that there is a shorthand for doing so. Why do you think symbols are so widely used as keys?

In [47]: 
{city: "Durham", state: "NC"}
Out[47]:
{:city=>"Durham", :state=>"NC"}

What did we do up there with person = ...?

That's a variable. We can assign names to values. Those names are called variables and you can use them in place of the value.

In [48]: 
letters = [:a, :b, :c, :d]
Out[48]:
[:a, :b, :c, :d]
In [49]: 
first_letter = letters[0]
Out[49]:
:a
In [50]: 
name = {:first => "John", :initial => first_letter}
Out[50]:
{:first=>"John", :initial=>:a}
